Harrison Network

Tunnel Service

An ngrok-like tunneling system which is configured using sish. You can request access by emailing tunnels@hrsn.net.

Available subdomains:

  • *.t.hrsn.net - HTTP, SSH, TCP tunnel support.
  • *.t.hrsn.dev - HTTP tunnel support.
  • Custom domains - contact us at tunnels@hrsn.net to add your own private tunneling domain.

Contact abuse@hrsn.net to report abuse of the tunneling system.
